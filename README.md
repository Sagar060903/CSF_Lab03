Aim of the Lab
To develop a clear understanding of control structures in Python by implementing conditional statements, loops, break and continue statements, and to learn how to define and use functions effectively.
📂 List of Programs
q1_positive_negative.py
q2_even_odd.py
q3_sum_of_numbers.py
q4_loop_example.py
q5_function_example.py
📝 Description of Programs
1. Positive or Negative
This program checks whether a given number is positive, negative, or zero using if-else statements.
2. Even or Odd
This program determines whether a number is even or odd using conditional statements.
3. Sum of Numbers
This program calculates the sum of numbers using a loop.
4. Loop Example
This program demonstrates the use of loops (for/while) and control statements like break and continue.
5. Function Example
This program shows how to define and call functions in Python to solve a simple problem.
